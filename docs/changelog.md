# Changelog

## 1.1

- Added Settings screen with a Persian calendar variant option:
  Solar Hijri (default) or Shahanshahi (Imperial, +1180 years).
- Setting persists across launches.

## 1.0

- Initial release.
- Today view with Persian and Gregorian dates and weekday.
- Gregorian ↔ Persian date converter.
- Set today's date (writes to the Flipper RTC).
